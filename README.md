## Projects

**🏟️ UABC Sports Center — Registration Platform**
Replaced 30–80 Google Forms per semester with one centralized web platform. Serves 100–500 students, faculty and external participants across 4 UABC campuses. Still in production.
`HTML · CSS · JS · PHP · MySQL`

<br/>

**📱 ServiApp — On-Demand Home Services**
Android app connecting users with local service providers — plumbers, mechanics, electricians. Real-time backend via Supabase.
`Kotlin · Android Studio · Supabase`

<br/>

**📦 Fito Scents — Inventory & Sales System**
Built a custom web tool to run my own fragrance resale business. Auto SKU generation, cost vs. price tracking, profit margins, multi-item cart, and accounts receivable with visual progress bars.
`HTML · CSS · JS · Bootstrap`

<br/>

**🤖 ML Explorer** *(in progress)*
Learning machine learning fundamentals. First goal: build a price prediction model using my own sales data.
`Python · scikit-learn · pandas`
